# 基于SSM的固定资产管理系统+毕业设计示例参考

#### Description
实现了资产类型管理，资产管理，资产借用，审核以及资产借用记录，同时实现了资产的维修和报废功能，图表实现了资产类别展示

技术栈+工具：SSM + JSP + Tomcat + IDEA2022 + Maven

管理员主要功能：用户管理、资产信息管理、借还管理、维修管理、报废管理

普通用户：借用登记、归还登记等
